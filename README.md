# 🌟 Professional Data Analytics Portfolio - Aya Khaled

A high-performance, full-stack dynamic portfolio designed for **Eng. Aya Khaled**, specialized in **Data Analysis**. This platform combines a sleek, data-driven frontend for visitors with a robust, secure administrative dashboard for complete content management.

## 🚀 Key Highlights

### 🎨 The Visitor Experience
- **Showcasing Expertise:** Dynamic sections for data projects, professional experience, and academic certifications.
- **Detailed Project Insights:** Dedicated pages for each project to dive deep into data methodologies and results.
- **Interactive Connectivity:** Direct integration with professional freelance platforms and social media.
- **Modern UI:** Fully responsive design built with **Angular 17+** and **Bootstrap 5**, ensuring a smooth experience on any device.

### 🛠️ The Admin Powerhouse (Dashboard)
- **Centralized Management:** A private dashboard to manage every single piece of data on the site (Projects, Skills, Experience, Freelance links).
- **Full CRUD Capability:** Real-time adding, updating, and deleting of records with instant reflection on the live site.
- **Secure File Handling:** Optimized logic for uploading and managing project screenshots and certificates.

---

## 🛡️ Technical Excellence & Security

This project isn't just a static site; it's a secured web application:
- **Silent Authentication:** Implemented a sophisticated **Double Token System (Access & Refresh Tokens)**.
- **Cookie-Based Security:** Sensitive session data is stored in **HttpOnly Cookies** to prevent XSS attacks.
- **Angular Interceptors:** Custom functional interceptors to handle token refreshing seamlessly without interrupting the user session.
- **Robust Backend:** Powered by **Node.js** and **Express**, using **Sequelize ORM** for reliable database communication.

---

## 🛠️ Tech Stack

- **Frontend:** Angular 17, RxJS, Bootstrap 5.
- **Backend:** Node.js, Express.js.
- **Database:** Sequelize (PostgreSQL for production).
- **Security:** JSON Web Tokens (JWT), Bcrypt encryption, Cookie-parser.
